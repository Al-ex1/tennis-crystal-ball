Best-of-5 vs Best-of-3, a completely different game

There are a lot of talks recently about tennis game formats and rules. We are all witnessing that best-of-5 format is slightly diminishing, despite most die-hard fans preferring it over the best-of-3.
There are no anymore Masters 1000 best-of-5 finals since 2007 and Davis Cup will move away from best-of-5 starting from this year.

The Difference

We all know that better players have more chances to win in the best-of-5 format compared to best-of-3. But how much more?
Using Markov Chains, one can express game, tie-break, set, best-of-3 and best-of-5 match probabilities.
Here are the formulas for best-of-3 and best-of-5 match win probabilities from set win probability:
m 3 = s 2 * (3 - 2 * s)
m 5 = s 3 * (10 - 15 * s + 6 * s 2)
However, it is not possible to mathematically solve above system of equations, i.e. it is not possible to create a mathematical expression that maps best-of-5 to best-of-3 match win probability. It is only possible to create a graph and a computer program to calculate it. Conclusion

From the graph and the table, it can be seen that if players are of similar strength and form, then there is very little difference between best-of-3 and best-of-5.
However, if one player has 65% or more chances to win, then the difference between best-of-3 and best-of-5 starts to be more and more significant.
So, best-of-5 is protecting higher-ranked players from upsets from lower-ranked players, while not affecting excitement in watching two similarly-ranked players clash.
As such, it is an invaluable tool for Grand Slam tournaments to ease the path of favorites towards the end rounds and keep having an ultimate entertainment in the semi-final and final matches.
